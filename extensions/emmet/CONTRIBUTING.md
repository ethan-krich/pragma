## How to build and run from source?

Read the basics about extension authoring from [Extending Pragma](https://code.visualstudio.com/docs/extensions/overview)

- Read [Build and Run Pragma from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Pragma
- Open the `extensions/emmet` folder in the vscode repo in Pragma
- Press F5 to start debugging

## Running tests

Tests for Emmet extension are run as integration tests as part of Pragma.

- Read [Build and Run Pragma from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Pragma
- Run `./scripts/test-integration.sh` to run all the integrations tests that include the Emmet tests.
